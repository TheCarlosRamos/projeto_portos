# Frontend - Sistema de Gestão de Processos e Metas

Frontend React + TypeScript para o sistema de gestão de processos e metas.

## Instalação

```bash
npm install
```

## Execução

```bash
npm start
```

A aplicação será aberta em http://localhost:3000

## Build para produção

```bash
npm run build
```

## Estrutura

- `src/components/` - Componentes reutilizáveis
- `src/pages/` - Páginas da aplicação
- `src/services/` - Serviços de API
